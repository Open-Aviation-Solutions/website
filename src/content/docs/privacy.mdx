## What this site collects today

This site is hosted on GitHub Pages. GitHub Pages records standard web server access logs (IP address, browser type, pages requested, timestamps). I do not control or access these logs directly — they are held by GitHub under [GitHub's privacy policy](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

Beyond that, the site collects nothing. No analytics. No tracking cookies. No forms that submit data anywhere.

## What's coming — and what I'll do before it ships

### Analytics

I plan to add an analytics tool to understand which parts of the site are useful. Before adding any analytics:

- I will update this policy to name the provider and describe what data it receives.
- I will update this policy to describe any opt-out mechanism available.

I won't ship analytics without updating this page first.

### Ads on learning-component pages

I may eventually add advertising to the free learning-component pages to offset
AI token and development costs. Before enabling ads:

- I will update this policy to name the ad network and explain that ad-served cookies and tracking are outside my direct control.
- I will add a cookie notice where required.

I won't ship ads without updating this page first.

If you have a question about privacy on this site, or want to know whether any
