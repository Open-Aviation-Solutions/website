title: Accessibility
description: Open Aviation Solutions' accessibility statement and known gaps.
---

As much as possible, I want Open Aviation Solutions to be usable by everyone. I've aimed for clear headings, sufficient contrast, and readable type throughout.

## Known gap: interactive learning components

Some of the interactive [learning components](/learning-components/) — including [Four Forces](/learning-components/four-forces/) — are built with WebGL and do not currently have equivalent text or non-visual alternatives. If you can't use a component due to an accessibility gap, please [create an issue for the specific component](https://github.com/Open-Aviation-Solutions/open-aviation-components/issues) describing how it could be improved.

## Other barriers

If you encounter any other accessibility barrier on this site, please [create an issue for the website](https://github.com/Open-Aviation-Solutions/website/issues) or [contact me](/contact/) and I'll address it.
